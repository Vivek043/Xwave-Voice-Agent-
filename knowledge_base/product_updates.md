# NovaCRM — Recent Product Updates

## April 2026 Release — v4.2

### New: AI-Powered Lead Scoring
We've added intelligent lead scoring powered by machine learning. NovaCRM now automatically scores your leads from 1–100 based on engagement patterns, email opens, website visits, and historical conversion data. Available on Professional and Enterprise plans.

### New: Bulk Email Templates
Create and save up to 50 email templates (Starter: 10, Professional: 30, Enterprise: 50). Templates support dynamic merge fields like {{first_name}}, {{company}}, and {{last_activity_date}}.

### Improved: Mobile App v3.0
The NovaCRM mobile app has been completely redesigned with a new bottom navigation, offline mode for viewing contacts, and push notifications for deal updates. Available on iOS 16+ and Android 12+.

### Fixed: Calendar Sync Issues
Resolved an issue where Google Calendar events were appearing with incorrect time zones for users in Asia-Pacific regions. If you experienced this bug, reconnect your calendar from Settings → Integrations.

## March 2026 Release — v4.1

### New: Workflow Automation Builder
Visual drag-and-drop workflow builder for automating repetitive tasks. Create triggers like "When a deal moves to Won → Send thank you email + Create invoice + Notify sales manager." Enterprise plan includes up to 100 active workflows.

### New: Custom Fields
Add up to 20 custom fields (Professional) or unlimited (Enterprise) to contacts, deals, and companies. Supported field types: text, number, date, dropdown, checkbox, and multi-select.

### Improved: Search Performance
Global search is now 3x faster with support for fuzzy matching. Search across contacts, deals, emails, and notes from the top search bar using Cmd+K (Mac) or Ctrl+K (Windows).

## February 2026 Release — v4.0

### New: Analytics Dashboard 2.0
Completely rebuilt analytics with customizable widgets, real-time data, and export to PDF/Excel. New metrics include Customer Lifetime Value (CLV), churn prediction score, and sales velocity tracking.

### New: Phone Integration
Make and receive calls directly from NovaCRM using our Twilio integration. Call recordings are automatically logged to the contact timeline. Available on Professional and Enterprise plans. Setup: Settings → Integrations → Phone → Connect Twilio.

### Security: Mandatory 2FA for Admins
All workspace administrators are now required to enable two-factor authentication. This can be configured with an authenticator app (Google Authenticator, Authy) or SMS verification.
